audio_ALC1220
============
OS X Realtek ALC1220 Onboard Audio

Supports:
ALC1220: device_id: 0x1168 (1220A)
ALC1220: device_id: 0x1220 (1220)

Update: Download/Double click and done.
See https://github.com/toleda/audio_RealtekALC
or https://github.com/toleda/audio_CloverALC

Use (xy is latest macOS release, i.e.: 10.12, xyz = 120, ) version supports all releases):
1. audio_cloverALC-xyz.command (recommended)
2. audio_realtekALC-xyz.command
